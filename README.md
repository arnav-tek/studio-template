# ✦ Nexus Labs - Premium AI-First Creative Engineering Studio ✦

An elite, high-performance website template for a digital agency specializing in premium design, custom software engineering, decentralized architectures, and advanced AI automation. Inspired by K3 Studios Dubai, this project showcases high-fidelity interactive elements, modern typography, and immersive animations.

Live preview / project clone designed for maximum aesthetic impact and seamless desktop/mobile experiences.

---

## 💎 Features & Visual Polish

* **Rich Interactive Visuals:**
  * **Glowing Hero Orb:** Dynamic mouse-following cursor glow effect that responds smoothly to user interaction.
  * **Grid Scan & Pulse Effects:** Custom laser pulses running across layout guidelines in both the hero and footer sections.
  * **Text Scramble Animations:** Highly polished hover effect that dynamically scrambles character structures before settling on text values.
* **Premium Scrolling & Motion Architecture:**
  * Powered by **GSAP (GreenSock Animation Platform)** with custom **ScrollTrigger**, **ScrollTo**, and **Observer** integrations.
  * Fully integrated **Lenis** smooth scrolling library for ultra-fluid viewport motion.
  * Custom page transitions featuring gold and dark panel swipes on link clicks.
* **Fully Responsive & Multilingual Ready:**
  * Clean, native CSS hamburger menu and responsive fullscreen overlay on mobile viewports.
  * Dynamic English (EN) and Arabic (AR) language switcher placeholders.
* **State-of-the-art Design System:**
  * Deep gold accents combined with an ultra-sleek dark glassmorphic palette.
  * Premium typography utilizing fonts like *Outfit* and *Inter*.

---

## 🛠️ Tech Stack & Libraries

* **Core Structure & Styling:** HTML5 & Vanilla CSS3 (Custom-built premium agency stylesheet `css/agency.css`)
* **Core Functionality & Animations:** Vanilla ES6+ Javascript
* **Animation Suite:**
  * [GSAP Core](https://gsap.com/)
  * [GSAP ScrollTrigger](https://gsap.com/docs/v3/Plugins/ScrollTrigger/)
  * [GSAP ScrollToPlugin](https://gsap.com/docs/v3/Plugins/ScrollToPlugin/)
  * [GSAP Observer](https://gsap.com/docs/v3/Plugins/Observer/)
* **Scroll Engineering:** [Lenis Smooth Scroll](https://github.com/darkroomengineering/lenis)

## 🚀 Getting Started & Local Development

To run the project locally, follow these simple steps:

### 1. Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### 2. Install dependencies & run development server
From the root of the project directory, run:

```bash
# Start local development server (runs http-server on port 3000)
npm run dev
```

Alternatively, you can open any of the `.html` files directly in your browser or run them using a web server extension like **Live Server** in VS Code.
